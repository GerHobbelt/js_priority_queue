h1. Priority Queue

A simple javascript priority queue using a binary heap. A compare function is used to sort the elements. The compare function has the same semantic like the ones used with @Array.sort()@.

h2. Synopsis

bc.. var pq = require('priority_queue');

// create a queue
var q = new pq.PriorityQueue(/* compare, array */);

// push some elements
q.push(42, 5, 23, Math.PI);

// pop
while (q.length > 0) {
  console.log(q.pop());
}
